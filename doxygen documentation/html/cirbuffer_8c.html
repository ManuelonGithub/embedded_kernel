Dequeues a character from a circular buffer.

Parameters
[in,out]bufferpointer to circular buffer being used.
Returns
char/byte that was dequeued from buffer.

This function may corrupt the buffer if the buffer is empty. Always make sure to check if buffer contains data before using this function, or use dequeue_s(), which will dequeue from the buffer safely, albeit with added overhead.
