char* itoa ( int  i,

)

Converts a signed integer into a string in base10.

Parameters
[in]iValue to convert.
[out]bufCharacter array to write number string to. Must be of size INT_BUF.`
Returns
Pointer to start of number in the character array. Function writes to buffer from the last character so the string visual fits wth how we view numbers. (largest decimal case to smallest). this means that the beginning of the buffer string may be invalid, so the returned pointer must be used to analyse the buffer data.

This function is based on the itoa implementation found in https://opensource.apple.com/source/groff/groff-10/groff/libgroff/itoa.c
