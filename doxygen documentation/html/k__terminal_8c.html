◆ COMMAND

const char* const COMMAND[]
Initial value:
= {
"SYSVIEW", "SETIO", "CLEARIO", "RUN"
}

Supported command keywords. Commands are case insensitive.

SYSVIEW: displays information about the current processes. IO: IO permissions command. Let's user configure or display IO permissions. RUN: places terminal process on the background and lets user processes run.
